# Goal Systems technical test

## Mock server setup
Install json-server
```
npm install -g json-server
```
Run mock server before starting the project
```
start_mock_server
```
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```
